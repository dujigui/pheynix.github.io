使用Hexo部署博客

2015-12-06

Hexo是一个静态博客框架，可以使用MarkDown写作然后方便地生成静态博客，部署的方式在官方其实已经很详细，网上也有很有详细的教程，这里主要记录一下部署过程遇到的坑。

1 Error: Deployer not found : git

执行$ hexo deploy 遇到Error: Deployer not found : git的错误，原因是没有安装deployer插件。

先执行rm -rf .deploy_git移除该插件现有的安装，然后npm install hexo-deployer-git --save安装插件。
