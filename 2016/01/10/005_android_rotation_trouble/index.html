Android旋转动画闪烁问题

2016-01-10

问题

项目需要实现点击按钮后旋转按钮的需求
在res/anim中新建anim_close.xml

和anim_open.xml

其中

  • fromDegrees表示动画开始时候的角度
  • toDegrees表示动画结束时候的角度
  • 上述角度叠加在ImageView的android:rotation之上；例如当前rotation是45°，fromDegrees是45°，那么动画从90°的位置开始播放。
  • 动画结束后恢复原样

开始的时候打算在**动画结束后修改调用ivShortcut.setRotation(45)旋转ImageView。但是由于动画结束首先恢复原样(0°)，然后才修改旋转角度，会造成闪烁的情况。

闪烁的屏幕录像

解决方案

先修改ImageView的角度为最终角度，然后启动动画，动画的fromDegrees值和toDegrees改为相应的值。

status fromDegrees toDegrees rotation description
打开－原 0 45 45 动画从0转到45，然后将ImageView的rotation设置为45
打开－新 －45 0 45 将ImageView的rotation设置为45，然后动画从相对当前位置的-45转到0
关闭－原 0 －45 0 动画从0转到－45，然后将ImageView的rotation设置为0
关闭－新 45 0 0 将ImageView的rotation设置为0，然后动画从相对当前位置的45转到0

最终能够得到正确的效果
